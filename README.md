# Portal Cautivo - Proyecto Linux

## Descripción
Portal cautivo en Ubuntu Server con:
- DHCP (dnsmasq)
- NAT (iptables)
- Autenticación web (Apache + PHP)
- Bloqueo de Instagram y ChatGPT

## Instalación

```bash
git clone https://github.com/tu-usuario/portal-cautivo.git
cd portal-cautivo
sudo chmod +x setup_portal_cautivo.sh
sudo ./setup_portal_cautivo.sh
